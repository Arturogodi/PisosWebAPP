Project Overview

For this TFM project, it was decided to create an End-to-End application developed in the Azure environment. The application provides an interactive interface with dynamic content, built using Django for both backend and frontend. The project also integrates Databricks for advanced analytics and machine learning model deployment.

Data Architecture

The data architecture of the application follows the **Medallion Architecture** for structured data management. This architecture involves three layers:

  • Bronze Layer: Raw, unprocessed data ingested from various sources.
  • Silver Layer: Cleaned, transformed data that is ready for analysis.
  • Gold Layer: Final dataset optimized for business analytics and machine learning use.

The datasets used in this application are stored in Azure Blob Storage in Delta format, enabling fast reads and writes, as well as compatibility with Spark for large-scale data processing.
